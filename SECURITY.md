# Security Policy

## Supported versions

仅支持最新公开源码版本和当前官方发布版本。

## Reporting

请不要为以下问题创建公开 Issue：

- API key、签名文件或其他凭据泄露；
- 认证绕过；
- 用户训练数据或备份数据暴露；
- 可以破坏同步数据完整性的漏洞。

请优先通过 GitHub Security Advisories 私下报告；如果该入口不可用，请通过仓库维护者主页提供的联系方式联系。

报告中不要包含真实训练数据、生产 API key、完整备份包或可直接访问的服务器地址。

## Operational scope

自托管同步服务器的安全问题应报告到 [yours-sync-server](https://github.com/Maqiaogongmin/yours-sync-server) 仓库。本仓库主要处理 App 客户端、Yours Vault 和同步客户端行为。
