# Contributing

## Current policy

当前优先接受 GitHub Issues，用于错误报告、功能建议、文档问题和兼容性反馈。

Pull Request 可以提交，但项目暂不承诺评审、合并或发布时间。请不要提交无法证明来源的代码、图片、字体、训练数据或其他资产。

## Before reporting

- 使用最新公开版本复现问题。
- 删除训练记录、服务器地址、API key、设备标识和其他个人信息。
- 写明平台、系统版本、App 版本和复现步骤。

## Licensing

除非另有明确书面说明，提交给本仓库并被接受的贡献将按照本项目相同的 `AGPL-3.0-or-later` 和 App Store additional permission 提供。

当前项目尚未采用 CLA 或版权转让协议。维护者正式扩大 PR 接受范围前，会重新评估贡献管理方式。
